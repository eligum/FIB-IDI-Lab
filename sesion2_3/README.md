# Sesion2_3

## Building
To compile use the following commands.
```console
$ qmake-qt5 Sesion2_3.pro
$ make -j
```

## Functionality
* Drag the mouse across the screen to rotate the model
* Scroll to zoom in and out
* Key::O to toggle between PERSPECTIVE and ORTHOGONAL projections
* Key::1 front view
* Key::2 top view
* Key::3 side view
